# Placeholder (fusion)

> Class: `RCG_PlaceholderSetting`

## Purpose
**Internal placeholder used by the card fusion system**. During fusion preview, nested structures (e.g. "Conditional with placeholders inside") are kept intact so players can see **where new effects will fit**. After fusion confirmation, placeholders are replaced with the actual children.

> [!IMPORTANT]
> **Don't create manually** — auto-generated by the fusion system at runtime. If you see one in your data, this card is in the middle of a fusion preview state.

## Key fields

| Inspector label | Required | Notes |
|---|---|---|
| **PlaceholderKey** | Yes | i18n key for hint text (default `CardFusionPlaceholder`, with PlaceholderIndex inserted). |
| **PlaceholderIndex** | Yes | Placeholder number (distinguishes multiple placeholders). |
| **PreviewDescription** | No | Custom preview text; non-empty overrides i18n result. |

## Behaviour
*   No-op at trigger time (pure placeholder).
*   Description prefers `PreviewDescription`; otherwise uses `PlaceholderKey` i18n result.
*   Acts as a leaf — `GetFusionCandidateSettings()` is empty (not a fusion candidate); `GetFusionBaseSetting()` returns self (not further replaced).

## Notes
*   **Don't author manually**: unless deliberately marking a "to-be-fused" hole.
*   **vs DiminishedPlaceholder**: this is the **fusion system's** neutral placeholder (blue frame); `RCG_DiminishedPlaceholder` is the **diminish system's** (red "Diminished" text).
*   **PreviewDescription is for testing**: handy override for visual mock-ups.

## Appendix: Programmer Reference

### A.1 Class info
*   **Path**: `CardGame/Assets/Scripts/RCG_Scripts/RCG_GameDatas/RCG_BattleSettings/RCG_PlaceholderSetting.cs`
*   **Inherits**: `RCG_BattleSetting`
*   **No i18n class key**: editor shows stripped name `Placeholder`

### A.2 Field map

| Code | Inspector | Type | Localize key | Notes |
|---|---|---|---|---|
| `m_PlaceholderKey` | PlaceholderKey | `string` | — | Default `"CardFusionPlaceholder"` |
| `m_PlaceholderIndex` | PlaceholderIndex | `int` | — | Inserted as `{0}` in template |
| `m_PreviewDescription` | PreviewDescription | `string` | — | Default empty; non-empty overrides |

### A.3 Key methods
*   **`GetDescription`**: non-empty `m_PreviewDescription` returns directly; otherwise `UCL_LocalizeManager.Get(m_PlaceholderKey, m_PlaceholderIndex)`.
*   **`GetDescriptionShort / GetShortName / GetDescriptionFormat`**: all delegate to `GetDescription`.
*   **`GetFusionCandidateSettings`** → empty.
*   **`GetFusionBaseSetting`** → `this`.
*   **`AddAction`** → empty.

### A.4 Cross-system interactions
*   **Fusion system's `GetFusionBaseSetting`**: the base class default returns `new RCG_PlaceholderSetting()`, so any subclass without overrides generates placeholders during fusion preview.
*   **`RCG_DiminishedPlaceholder`**: subclass for the diminish system.
